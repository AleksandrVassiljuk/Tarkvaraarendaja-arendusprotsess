Programmeerimise alused

See aine õpetas mulle programmeerimise põhitõdesid. Õppisin, kuidas kirjutada lihtsaid programme Python keeles. Sain aru, kuidas loogika ja kood koos töötavad.

Teemad

IDE ja Python sissejuhatus

Õppisin, mis on IDE ja kuidas seda kasutada. Tutvusin Python programmeerimiskeelega. Kirjutasin oma esimesed lihtsad programmid.

Voodiagramm

Õppisin, kuidas teha voodiagramme. Need aitavad programmi loogikat paremini mõista. Joonistasin hommikuste tegevuste algoritmi.

Arvude teisendus

Õppisin teisendama arve erinevate süsteemide vahel. Näiteks kahend- ja kümnendsüsteem. See arendas minu loogilist mõtlemist.

Järjend ja sõne

Õppisin, mis on järjend ja sõne. Kasutasin neid andmete hoidmiseks programmis. Harjutasin nende kasutamist ülesannetes.

Tingimuslaused

Õppisin kasutama if-lauset. See aitab teha otsuseid programmis. Näiteks kontrollida tingimusi.

Tsüklid

Õppisin while ja for tsükleid. Need aitavad korduvaid tegevusi teha. Kasutasin neid erinevates ülesannetes.

Funktsioonid

Õppisin, mis on funktsioon. Funktsioon aitab koodi lihtsamaks teha. Kasutasin funktsioone ülesannetes.

Matemaatilised avaldised

Õppisin kasutama matemaatilisi tehteid koodis. Näiteks liitmine, lahutamine ja korrutamine. Lahendasin ülesandeid ja teste.

Koodistiil

Õppisin kirjutama korrektset koodi. Hea kood on loetav ja arusaadav. See on oluline programmeerimises.

Git ja commit

Õppisin kasutama Git versioonihaldust. Tegin commit’e ja salvestasin muudatusi. See aitab tööd paremini hallata.

PyCharm ja versioonihaldus

Õppisin kasutama PyCharm programmi. Seadistasin Git-i oma projektis. See teeb programmeerimise mugavamaks.

Tarkvara arendusprotsess

Minu portfoolio sai alguse tarkvara arendusprotsessi ainest. Seal õppisin, kuidas projekte planeerida ja teha. See aitas mul luua oma ainemapi.
