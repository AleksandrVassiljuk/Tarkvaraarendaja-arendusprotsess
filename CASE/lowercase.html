LowerCASE vahendid

Mis on LowerCASE vahendid?

LowerCASE vahendid toetavad tarkvara arenduse hilisemaid etappe — koodi kirjutamist, testimist, versioonihaldust ja süsteemi hooldust.

Mille jaoks neid kasutatakse?

  • Koodi loomine ja silumine
  • Automaatne koodi genereerimine
  • Testide läbiviimine
  • Koodirepositooriumi haldus

Milliseid LowerCASE vahendeid olen kasutanud?

  • Visual Studio Code
  • Git
  • Postman
  • Selenium

Näited vahenditest, mida pole kasutanud

1. JetBrains Rider

Rider on .NET ja C# arenduskeskkond, mis sisaldab silurit, koodi analüüsi ja testimise tööriistu.

2. Jenkins

Jenkins on CI/CD tööriist, mida kasutatakse automaatse testimise ja koodi ehitamistöövoogude jaoks.
